牛逼！川普对美国最为腐败的机构退休军人动手了！

发表于： 2025-03-07 分类于： 价值
字数： 4106 阅读：≈ 9分钟 浏览：

抱歉，我为我看错了川普而道歉。

我之前说，修理美国教育部和国际开发总署，只是川普象征性裁员行动，他要敢动退伍军人事务部和国防部的人，才叫牛逼。

意思是，我认为他不敢。

但这一次川普真的这么干了，尽管我经常骂川普在国际贸易问题上是个SB，但我要承认，在对美国政府内部的改革上，他是个当之无愧的英雄人物。

据美联社周三获得的一份内部备忘录，美国退伍军人事务部（Department of Veterans Affairs）正计划进行重组，其中包括从这个为数百万退伍军人提供医疗保健和其他服务的庞大机构中裁减 80,000 多个工作岗位。

这一刀，力度不小，一下裁减八万名员工。

这说明，川普是来真的了。

要知道，这个部门，在过往是共和党和民主党建制派们不敢动的部门，他代表着美国最大的政治正确之一，那就是对退伍军人的“照顾”。

这个部门有多少人呢？

说出来吓死你，48万人。

退伍军人事务部的幕僚长克里斯托弗·西雷克 （Christopher Syrek） 周二告诉该机构的高层官员，它的目标是裁减足够的员工，以恢复到 2019 年略低于 400,000 人的员工水平。

也就是裁减了八万人以后，依然有四十万名员工。

美国光是为美国退伍士兵服务，就组建了一个多达48万人的庞大联邦机构，想想都不可思议啊。

这个机构干什么呢？

最主要的工作，就是协助退伍军人们拿退伍福利。美国有多达近一千万的退伍军人，光是一年要处理他们的医保预约问题就高达1.28亿次。

那我们进一步要问，为什么一个这样的机构需要48万人？

我们不讨论退伍军人是否要有福利的问题，就假定是必须的，在我看来，这个部门八十个人就够了。

你将要给退伍军人的钱，直接由这八十个人通过与银行合作，打过去就行了。有几个数据处理高手，就能完美解决这个问题。

那这个机构究竟在干什么？

要理解这个机构的工作啊，你需要一定的年纪，经历过中国四十年前的大国企，你就明白了。

在四十多年前，中国的大国企，有一种现象，叫国企办社会。

一个较大的国企里，可以说职工要的一切需求都有，包括从托儿所，幼儿园，小学到高中的学校，有食堂，有医院，有澡堂，基本上，国企职工不出厂区，一切生活都能安排完。

美国退伍军人事务部，就是一个接近于中国国企办社会的机构。

这个机构的主要职能我列一下，你就明白了。

VA（退伍军人事务部）为退伍军人提供了"从摇篮到坟墓"的全方位服务：

  • 医疗保健（从初级保健到专科护理）
  • 心理健康服务
  • 长期护理和养老院服务
  • 教育培训和就业援助
  • 住房支持
  • 经济补偿
  • 殡葬服务

但之所以需要48万人，那就是他们在执行“按需分配”的福利支配 方式 ，也就是共产主义的分配模式。

你看提供这一切服务，就需要自己办医院、办养老院吧，还需要审核对方的身份，确定对方的情况，以确定对方是否在这个时候需要福利支持。

这得养多少人呢？

VA下辖了一个美国最大的综合医疗系统之一，需要大量医护人员：

  • 约有35万员工（占VA总人数的80%以上）在VHA工作
  • 管理171个医疗中心和1000多个门诊设施、130个疗养院
  • 每年服务约900万登记退伍军人
  • 2023财年，VA医疗系统处理了1.17亿次门诊、约90万次住院和约12.1万次手术

VA医疗系统的规模相当于全美50个中等规模医院系统的总和。

当然除了医疗，还有一系列福利支出要管理。

VA下属一个机构叫退伍军人福利管理局，它的职能是：

  • 伤残补偿：为服役期间受伤或疾病而致残的退伍军人提供每月补助

  • 退伍军人养老金：为低收入战时服役退伍军人提供经济支持

  • 教育福利：包括"9/11 G.I. Bill"等，资助退伍军人接受高等教育

  • 职业康复和就业：帮助残疾退伍军人找到和保持适当工作

  • 住房贷款担保：帮助退伍军人获得购房贷款

  • 人寿保险：提供多种保险选择

  • 幸存者福利：为死亡军人的配偶和受抚养人提供支持

    你看，无所不包吧。

这个机构是一个吞金兽啊。

在创立之初，他的预算只占美国联邦总预算的0.8%。

现在呢，达到了5%，超过了3000亿美元。

更让你想不到的是，美国各州也有专门的退伍军人服务机构，支出也高达400亿美元以上。算下来，美国所有政府为退伍军人支出的费用高达3400亿美元。

并且，我还要告诉各位，这一笔钱不计入美国军费支出之内。要知道，美国光是军费支出又高达8000多亿美元。

更为搞笑的是，这是在一个和平的时代，美国基本上没有外部威胁的时代，光光是退伍军人的支出，就已经超过了全球除美国以外所有国家的军费支出了。

这么庞大的支出，真的是对退伍军人的优质服务吗？当然不是。

纽约时报记者大卫·菲利普斯曾因为系列报道揭露了凤凰城退伍军人等候名单丑闻，赢得了普利策奖。

这个记者在调查发现，2014年亚利桑那州凤凰城VA医疗中心的员工伪造等候记录，掩盖退伍军人长时间等待就医的事实，进一步的调查发现，至少有35名退伍军人在等待医疗服务期间死亡。

丑闻发布后，全国调查，发现全国范围内的VA设施普遍存在类似问题。

为此美国国会的问责办公室(GAO)发布了一个报告，报告显示一些退伍军人等待初次预约长达115天。

也就是说，虽然花了全世界最多的钱，虽然有几十万员工，但是，军人们就医依然要花大量的时间等待，甚至在等待中死去。

除了医疗等待这种问题外，有几万人在做行政作业，但伤残赔偿这种事，也是大量积压。

2013年，VA积压的伤残索赔超过60万份，一些退伍军人等待决定长达数年。至2023年，尽管有所改善，但仍有超过40万份积压索赔。平均处理时间约为125天，远高于VA设定的125天目标。而上诉案件的处理时间更长，平均超过3年。

2018年，VA在压力下说要搞电子化办公，就搞了一套电子健康记录系统，预算为100亿美元。一百亿美元开发一套软件，那是财大气又粗啊。

但这样的软件在部署中遇到严重问题，包括用户界面困难、系统崩溃和数据迁移错误。

你知道么，微软投一百亿搞出来轰动全球的人工智能软件了。

但你更想不到的是，这套软件上不了线，于是推迟，要求增加预算，预算是多少呢？翻个倍，200亿美元，2030年上线。

老天爷呀，收了这帮子妖孽吧，一套数据库软件，交给印度阿三，搞个几十个人，几个月就给你干出来了呀，给几百万美元他们都要发财了。

这倒好，这玩意花二百亿美元，还得花12年的开发时间。

搞不好，2030年上线，还是不能正常运转。

了解过往国企大社会服务机构效率的人，就能明白这是为什么了。

这就是一种纯纯的计划经济下的生产供应模式，他不以利润为目标，在这个体系中，基本消失了价格信号，完全由政府以下指令的方式为退伍军人生产具体的福利供给。

服务者无一例外，都属于政府公务员，他们行动的动因，来源于上级的命令，是纯粹的官僚机构。

这种机构低质、低效就是一个必然。

美国邮政虽然说是美国的国企，长年亏损，但起码这个机构还是收钱的，还有一丢丢服务消费者的属性，但VA这个机构，就是全世界最烂的机构——计划经济运行体制，他不存在经济计算，根本不需要考虑利润，也不需要考虑服务消费者。

这个机构的丑闻不断：

2015年，奥罗拉科罗拉多州的VA医疗中心建设项目预算从6亿美元膨胀到17亿美元以上

2016年的审计发现，VA在一年内错误支付了约50亿美元

2018年，国会问责办公室报告发现，VA药品采购浪费超过10亿美元

但没关系，拜登在任上，让这个机构的经费和人员达到历史最高点。不够花就多花点，效率不够，那肯定是人手不够嘛 。

可是，就这么一个腐败到顶的机构，两党不敢乱动。

只有川普这种政客敢说且敢干。

特朗普在2016年竞选期间将VA的改革作为主要议题，称其为"政府最腐败的机构"，但第一个任期，在各种压力下，他没有进行改革。

VA的预算是国会定期争论的焦点：

2023财年，VA预算约为3010亿美元，比2001年的450亿美元增长了近7倍

但两党都不敢直击问题本质。

共和党通常批评管理效率低下而不是资金总量。

民主党往往支持增加资金，特别是用于扩大服务范围和提高质量

2022年，《焚烧坑法案》(PACT Act)提出，要求为接触有毒物质的退伍军人提供扩展医疗保障，预计在未来十年花费2800亿美元。

一些共和党议员最初反对该法案，担心成本上升，但在强烈的公众压力下最终支持了它。

美国在对待退伍军人问题上，可是一个无比敏感的问题。

在美国盖洛普、哈里斯等咨询公司“最受尊重职业”的民意调查中，美国军人连续多年位居前列。

我记得矮大紧曾经说过，美国是他见过军人地位最高的国家。

美国有《退役军人优先权法》和《双重补偿法》，规定退役军人和符合一定条件的军人配偶或遗属在政府部门和私有企业就业方面享有优先权；退役军人可优先回到服役前的工作岗位。

裁减职员时，退役军人有保留工作的权利，雇主不得以任何理由辞退。

若企业欲承接联邦政府项目，则必须表明已对退役军人积极招募且没有歧视，否则将失去与政府合作的机会。

美国政府还将每年的11月11日定为“退伍军人节”，放假一天，各城市举行盛大的庆祝游行，退役军人身着军装，参加游行，市长还要接见，并授予纪念勋章。

除了这几十万联邦工作人员外，还有遍布各地的非政府退役军人服务机构，则人数不可胜计。就是一个普通小镇的“退役军人协会”影响也很大，当地政府、社区和驻军都会主动接洽联络，协会成员经常组织各种聚会和活动，大日子还会盛装出场。

川普当下还是对官僚机构开刀，但这已经是改革的第一步了。

这类的机构，实际上是可以全部砍掉的，真要发福利，直接发钱到退伍军人账户就可以了，让他们自己采购医疗服务和教育服务，何必要政府建个医疗系统呢？

流传于苏联带头推动的国企办社会机制，居然还在美国存在，且这么大的规模，可谓是对美国这种号称资本主义国家最大的讽刺了。

要知道，这个部门的开销，已经超过了印度整个政府的全年经费支出了。

川普之所以在这件事上，我愿意称之为英雄，哪怕他在贸易战和阻止移民问题上骂他SB，但不能让我放弃我对此事的赞美，因为这是冒天下之大不韪、冲撞美国社会政治正确的行动。

因为，政治正确，就是一种束缚人们行动的观念之网，敢于挑战者，在这个时代都不多的，都需要冒着骂名，被人家视为没有“良知”。

川普如能持续改革这个机构，削减美国财政支出，从这一点上，从二十世纪初以来，是没有人敢干这件事的。

他自恋一点，也是可以接受 的。
