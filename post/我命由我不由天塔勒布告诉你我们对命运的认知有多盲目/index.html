我命由我不由天？塔勒布告诉你，我们对命运的认知有多盲目！

发表于： 2025-04-21 分类于： 价值
字数： 2280 阅读：≈ 5分钟 浏览：

“我命由我不由天”——这句出自电影《哪吒之魔童降世》的台词，一经推出便火遍大江南北。

它传递了一种不向命运低头的豪情，仿佛在告诉我们：只要有决心、有行动，命运就掌握在自己手中。这句话激励了无数人，让大家相信个人的选择和努力可以改变一切。

然而，当我们停下来仔细想想，这句话真的完全正确吗？决定一个人命运的，真的是只有自己的意志吗？

恐怕没那么简单。

生活中，集体观念、自然规律，甚至那些突如其来的意外事件，都在影响着我们的命运。

今天，我们就以这句台词为切入点，借助哲学家兼作家纳西姆·塔勒布的智慧，来分析“我命由我不由天”这句话的不足之处，并提醒大家：只有尊重自然规律，才能真正迎来好运。

一、随机性：命运的不可控一面

塔勒布在《随机漫步的傻瓜》中告诉我们，生活不像数学题那样有固定的答案，而是充满了随机性。这些随机事件往往是我们无法预测的，却能直接影响一个人的命运。

1.1 努力不等于结果

我们常听人说“努力就会有回报”，这话听起来很励志，但现实却没那么公平。假设有两个年轻人，都在拼命学习准备高考。

一个家里有钱，请得起名师辅导，另一个却只能靠自己摸索。他们的努力程度可能一样，但起点不同，结果很可能也不同。

这种差异，不是个人能完全决定的，而是由出生环境这种随机因素造成的。

再比如职场上的竞争。两个同事能力差不多，都想升职加薪，但最后机会可能因为老板的个人喜好、公司的一次裁员，或者市场的突然变化，而落到其中一个人的头上。

这种时候，努力只是基础，运气也很重要。

1.2 意外打乱一切

更别提生活中那些更大的随机事件了，比如自然灾害、意外事故。2020年的新冠疫情就是个活生生的例子。

疫情一来，无数人的生活被打乱：有人丢了工作，有人失去了亲人，有人苦心经营的小店一夜关门。

这些都不是个人努力能改变的，而是随机性带来的冲击。

塔勒布说过：“我们总觉得自己能控制一切，但其实随机性无处不在。”

“我命由我不由天”听起来很振奋，可它忽略了这些我们无法掌控的因素。命运，不完全是自己说了算。

二、黑天鹅事件：小概率，大影响

在《黑天鹅》这本书里，塔勒布提出了一个很有名的概念——“黑天鹅事件”。所谓黑天鹅，就是那些概率极低、但一旦发生就影响巨大的事件。这些事件没法预测，却能彻底改变一个人的命运。

2.1 无法预知的打击

想想2008年的全球金融危机。那时候，很多普通人一夜之间丢了工作，存款化为乌有，甚至房子也被银行收走。

这场危机来得太突然，连专家都没完全料到，更别说普通人了。这种黑天鹅事件，完全超出了个人的控制范围。

再比如生活中的突发状况。一个人可能平时身体很好，每天锻炼，却突然因为一次车祸受了重伤，从此生活完全改变。

这种事的概率很小，但一旦发生，个人的努力和选择就显得无力了。

2.2 集体观念的盲点

更有意思的是，我们的集体观念有时候会让我们忽视这些风险。社会总教我们“多努力就能成功”，让我们觉得未来是可以靠自己规划的。

可这种想法，往往让我们对黑天鹅事件毫无准备。等到事情真的发生，才发现自己根本应付不下了。

塔勒布提醒我们，黑天鹅事件虽然没法预测，但我们可以让自己更强韧一点。

这告诉我们，光靠“我命由我”的信念是不够的，还要正视那些不可控的因素。

三、反脆弱性：从不确定性中成长

到了《反脆弱》这本书，塔勒布提出了一个更厉害的想法——“反脆弱性”。啥叫反脆弱？就是不仅能扛得住随机事件的打击，还能从中变得更强。这其实给我们指了一条路：与其一味抗拒命运，不如学会跟不确定性相处。

3.1 尊重自然规律

自然规律是客观存在的，谁也绕不过。比如经济有周期，市场有波动，这些都不是个人能改变的。如果一个人能认清这些规律，顺势而为，就更容易站稳脚跟。比如说，做生意的人如果能看懂市场趋势，及时调整策略，就比一味硬干的人更有机会成功。

反过来，如果只信“我命由我不由天”，不顾规律地蛮干，反而容易摔跟头。比如有些人炒股，觉得自己能逆势赚钱，结果市场一跌，全赔进去了。这就是不尊重自然规律的下场。

3.2 集体观念的力量

集体观念其实也是一种规律。社会现状是集体观念的投射，社会怎么发展，文化怎么变化，都会影响我们的命运。

如果大多数人都不只持市场经济，都反对私有产权制度，个人努力的天花板就可能被集体的观念锁死。个人要想挣脱这个命运，必须付出巨大的成本。

当初，那些被关牛棚，甚至被整死的知识分子，无论个人学识和地位有多高，都免不了悲惨的命运。

个人的选择当然重要，但大趋势也得看清楚。

3.3 怎么变得反脆弱？

塔勒布给了几个实用的建议。比如，生活中多试错，别怕失败。失败了就学点东西，下次做得更好。比如找工作时，多学几门技能，别只靠一个本事吃饭。这样即使遇到意外，也不至于一下垮掉。

还有个办法是分散风险。就像投资别把钱全放一个股票上，生活中也别把希望全寄托在一件事上。这样即使有黑天鹅事件砸下来，也不至于全军覆没。

四、结语：好运从敬畏自然规律开始

“我命由我不由天”是一句很燃的话，它让我们看到了个人奋斗的价值。但如果我们只看到个人努力，忽视了随机性、黑天鹅事件和自然规律，那就太片面了。

命运从来不是一个人的独角戏，它是个人选择、集体观念和外部环境共同作用的结果。

塔勒布用他的随机性三步曲告诉我们：这个世界充满了不确定性，但我们不是只能被动挨打。

只要尊重自然规律，学会跟随机性共存，甚至利用它，我们就能在不确定的世界里找到自己的位置。

所以，与其一味喊“我命由我”，不如多点敬畏，多点智慧。既要有改变命运的勇气，也要有接受现实的胸怀。

好运，不是靠蛮力抢来的，而是靠尊重规律、拥抱变化得来的。

愿我们都能在人生的路上，既有豪情万丈，也有脚踏实地，最终遇见属于自己的好运。
