真正的高手，都是能拿捏人性的

掌握人性是成为高手的关键，需结合谋略、处世、心计、识人、智慧来理解人性。高手懂得隐藏实力，等待时机，以不露声色的策略取胜。在人际交往中，保持真诚和善意，细心体贴，避免触碰他人底线，树立良好形象。要懂得控制信息，保持神秘，展现智慧和谨慎。识人需审时度势，保持独立判断，避免轻信和受骗。智慧在于平衡理性和感性，专注等待机会，展现耐心和智慧。成为高手需深刻理解和运用人性。

在人际交往的复杂世界里，掌握人性，洞悉他人的心思和动机，是成为真正高手的重要一环。从谋略到智慧，每个领域都需要对人性有深刻理解和灵活运用。以下将结合谋略、处世、心计、人性、识人、智慧六个方面展开探讨，揭示高手在如何拿捏人性上的精妙之处。

谋略

1、闭嘴的鱼，最不容易被鱼钩钩住。

2、不要因为一时投缘，就轻易亮出你的底牌。

3、能控制住自己嘴巴的人都很厉害，不管是吃饭还是说话。

4、挑你毛病的人，只是想立威，并不是你真的有毛病。

5、带着答案问你问题的人，他要的不是答案，而是你的把柄。

6、不要急于报仇，烂掉的水果会自己从树上掉下来。

高手懂得隐藏实力，等待时机，以不露声色的策略取胜。建议在面对挑战时，保持冷静，不轻易暴露底牌，灵活运用谋略应对各种情况。

处世

1、别人请客，你不要带朋友过去。

2、去别人家吃饭，一定不要空手去。

3、亲戚之间接触越多，矛盾越多。

4、去别人家串门，要晚到不早到。

5、借别人钱，还钱的时要带点东西。

6、不加朋友老婆微，存个电话就行。

7、没有实力的时候，不要讨好别人。

8、保持边界感，不随便跟别人诉苦，没有真正的感同身受。

处世之道在于尊重礼节，回报感恩，保持适度的距离和独立性。建议在人际交往中，保持真诚和善意，细心体贴，避免触碰他人底线，树立自己的良好形象。

心计

1、别人不开口请你帮忙，尽量不要主动帮忙。

2、做人不能太老实，再好的朋友也有可能出卖你。

3、人际交往的高段位技巧：热情，大方，一问三不知。

4、不要告诉别人你的计划，直接让他们看结果。

5、你真正要做的事，连神明都不要讲，安静的做，成功了再说。

在人际交往中，要懂得控制信息，保持神秘，展现自己的智慧和谨慎。建议保持沉默和保守，不轻易透露个人信息和计划，以此保护自己的利益和安全。

人性

1、嘴巴甜的人尽量不要深交。

2、经常诉苦的人一定是废物。

3、自来熟的人一般不值得信任。

4、强大的人不会说悲惨经历。

5、满囗仁义道德的人一般都自私。

6、情绪稳定的人格局都很大。

7、越乖的孩子越没出息。

8、越调皮的孩子越聪明。

9、你越信任的人伤你越深。

辨别人性真伪，避免轻信，保持独立思考和冷静应对。建议在交往中保持警惕，审时度势，辨别他人的动机和性格，避免受骗受伤。

识人

1、你送他一颗糖，他立马还给你一颗枣的人，可以合作，但不适合做朋友。

2、嘴巴太快的人，往往没有什么城府，嘴巴太甜的人，不可以深交。

3、话少的人，往往是两个极端，要么真的简单，要么深不可测。

4、性格写在脸上的人，人都不会太差，而能够控制情绪的人，都不是一般人。

5、开口就说我有个朋友很厉害的。这种人往往没什么真本事。

识人之道在于审时度势，保持独立判断，避免轻信和受骗。建议深入了解他人，审慎选择交往对象，不轻易相信表面现象，保持独立思考和判断。

智慧

1、人性的扭曲之处就是：你越有霸气、底气，越会有人尊重你。你越是客气，和气越会有拿你不当回事。

2、与任何人的关系，你若想留住，就别它看得太重。只要看得太重，就必然会失去。

3、凡事从利益的角度，去衡量、去思考，你就会变得越来越聪明;如果从情感的角度考虑问题，就会越来越笨。

4、当你对一切都无所求，心无旁骛地把自己的事情做好的时候，你想要的自然就都来了。

智慧的体现在于平衡理性和感性，坚持专注，等待机会，展现出自己的耐心和智慧。建议从利益和情感的角度综合考量问题，保持冷静和耐心，勇敢追求目标，展现出自己的智慧和勇气。

总结：

真正的高手不仅在技术和能力上出色，更在对人性的把握和运用上具备独特的智慧和见识。从谋略到智慧，每一个领域都需要对人性有深刻的理解和敏锐的洞察力。只有懂得把握人性，才能在人际交往和挑战中游刃有余，展现出真正的高手风范。愿我们都能在人性的丛林中披荆斩棘，成为真正能够拿捏人性的高手。

格局

  1. 要懂得扮猪吃老虎，平时不要显摆自己的聪明。在人际交往中，有时候保持低调和谦逊的态度能够更好地获得他人的尊重和支持。

  2. 要学会伪装自己的情绪，不论和谁交谈都要保持随和的态度。懂得伪装自己是一种重要的生存技能，能够帮助我们在不同的环境中更好地适应和融入。

  3. 暗中布局，谋事先谋人，人通了事就好做了一出手就是定局。在处理事务时，我们需要善于观察和分析，提前做好规划和准备，同时也要注重与他人的沟通和合作，以便更好地达成目标。

  4. 学会识人、夸人、用人，让天下英才为你所用，切忌太过自我。在团队管理和人际关系中，我们需要善于发现和利用他人的优点和潜力，给予他们适当的赞扬和鼓励，同时也要保持谦虚和包容的心态，避免过于自我中心。

谋略

  1. 闭嘴的鱼，最不容易被鱼钩钩住。在人际交往中，有时候保持沉默和谨慎的态度能够更好地保护自己的利益和避免陷入困境。

  2. 不要因为一时投缘，就轻易亮出你的底牌。在与他人交往时，我们需要保持一定的保留和警惕，不要轻易暴露自己的弱点和秘密，以免被人利用或伤害。

  3. 能控制住自己嘴巴的人都很厉害，不管是吃饭还是说话。善于控制自己的言行举止是一种重要的修养和智慧，能够帮助我们更好地处理人际关系和应对各种挑战。

  4. 挑你毛病的人，只是想立威，并不是你真的有毛病。有些人可能会故意找茬或挑剔你的缺点，目的只是为了显示自己的权威和优越感，并非真正关心你的问题。

  5. 带着答案问你问题的人，他要的不是答案，而是你的把柄。有些人会故意提出问题来引诱你透露一些敏感信息或秘密，以便利用这些信息对你施加影响或威胁。

  6. 学会分钱，学会这一点，人才皆为你所用。在团队合作和管理中，懂得合理分配利益和奖励是吸引和留住人才的关键。只有给予他人适当的回报和认可，才能激发他们的积极性和创造力。

驭人

  1. 一个样样都不如你的人，你对他越尊重他越会报答你。在与他人交往时，我们应该保持谦虚和尊重的态度，不论对方的能力或地位如何，给予他们应有的尊重和关注，这样往往能够获得更多的支持和帮助。

  2. 如果你想了解一个人是否值得交往，记住，先试着冷落他一段时间。通过观察他在被冷落后的反应和行为，我们可以更好地判断他的真诚度和可靠性，从而决定是否与他建立深入的人际关系。

  3. 对老人要吹捧，人老了退休了，就只剩下那点回忆了，多捧着他，他就能为你所用。老年人经历了丰富的人生历程，他们渴望得到尊重和关注。通过给予他们适当的赞美和关心，我们能够建立起良好的互动关系，并从中获得他们的智慧和支持。

  4. 如果想知道一个人是小人还是伪君子，你可以先主动牺牲两次，看看他是如何对待你的牺牲的。通过观察他在你遇到困难或需要帮助时的态度和行动，我们可以更好地判断他的人品和价值观，从而决定是否与他建立信任和合作的关系。

处世

  1. 别人请客时，不要带朋友过去。尊重主人的安排和意愿，避免给他带来额外的负担或困扰。

  2. 去别人家吃饭时，一定不要空手去。带上一些小礼物或食物，以表达对主人的感激和尊重。

  3. 亲戚之间接触越多，矛盾可能越多。在处理亲戚关系时，要保持适度的距离和尊重，避免因为琐事而产生不必要的纷争。

  4. 去别人家串门时，要晚到不早到。遵守主人的时间安排，不要给对方造成不便或压力。

  5. 借别人钱时，还钱时要带点东西。除了归还本金外，可以带上一些小礼物或感谢的话语，以示对对方的感激和诚意。

  6. 不加朋友老婆微信，存个电话就行。尊重他人的隐私和个人空间，避免越过界限或引起误会。

  7. 没有实力的时候，不要讨好别人。保持真实和真诚的态度，不要为了取悦他人而做出虚假的表现或承诺。

  8. 保持边界感，不随便跟别人诉苦没有真正的感同身受。在与他人交流时，要注意控制自己的情绪和言辞，避免过度暴露个人隐私或给他人带来负面影响。

人性

  1. 嘴巴甜的人尽量不要深交。善于花言巧语的人往往隐藏着其他目的，不可轻信其言行。

  2. 经常诉苦的人一定是废物。过度抱怨和消极情绪表明缺乏解决问题的能力和积极心态。

  3. 自来熟的人一般不值得信任。过于主动亲近他人可能是出于某种目的，需要保持警惕。

  4. 强大的人不会说悲惨经历。成功者通常不会过多提及过去的失败或困难，而是专注于未来的发展和成长。

  5. 满口仁义道德的人一般都自私。口头上的道义并不能代表真正的善良和正直，需要通过行动来验证。

  6. 情绪稳定的人格局都很大。能够控制情绪并保持冷静的人通常具有更高的智慧和决策能力。

  7. 越乖的孩子越没出息。过于顺从和依赖他人可能导致缺乏独立思考和自主性。

  8. 越调皮的孩子越聪明。调皮捣蛋的行为可能是孩子探索世界和表达自我的方式，同时也反映了一定的创造力和智慧。

  9. 你越信任的人伤你越深。在人际关系中，过于相信他人可能导致被背叛或伤害的风险增加。

智慧

  1. 人性的扭曲之处就是：你越有霸气、底气，越会有人尊重你。你越是客气，和气越会有人拿你不当回事。在人际交往中，保持自信和坚定的态度能够赢得他人的尊重和认可。

  2. 与任何人的关系，你若想留佳，就别把它看得太重。只要看得太重，就必然会失去。过度依赖或过度在意他人的看法和态度，可能会让我们失去自我，并导致关系的破裂。

  3. 凡事从利益的角度去衡量、去思考，你就会变得越来越聪明；如果从情感的角度考虑问题，就会越来越笨。理性地从利益出发进行决策和行动，能够帮助我们做出更明智的选择。

  4. 当你对一切都无所求，心无旁骛地把白己的事情做好时，你想要的自然就都来了。专注于自己的目标和努力，不被外界干扰和诱惑所动摇，才能够实现自己的理想和愿望。

  5. 和人相处，你越是随意越是高冷反而人家会越爱你；当你开始上心听话，他就反而对你忽冷忽热。保持一定的距离和独立性，不过分迎合他人的期望和要求，能够吸引到真正欣赏和珍惜你的人。
