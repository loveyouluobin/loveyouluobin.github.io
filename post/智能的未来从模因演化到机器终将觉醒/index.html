智能的未来：从模因演化到机器终将“觉醒”

发表于： 2022-03-05 分类于： 价值
字数： 19524 阅读：≈ 39分钟 浏览：

本文，将在宇宙熵增的视角下，通过对基因、模因、智能三者从古至今演化逻辑的分析，从而按图索骥地推演出智能未来可能的进化路径，并且在这条路径之上，我们还会看到人类未来的演化命运，以及人类与人工智能、人类与机器智能所纠缠在一起的深刻关联。

需要指出的是，本文是在**“机器主义”理念下，对人类命运的“悲观预测”**，其结果是反直觉、反情感、反希望与反人性，但合乎逻辑的——如果对人类自身充满光明的愿景，如果对人类未来充满美好的愿望，那么本文所传递的信息，将可能会引起强烈的不适。

换言之，一切都在于选择，而选择是来自于，基因本能与模因智能的博弈，那么最高级的选择，就是可以选择自己的情绪——如果不是**“无情”的人，或不是可以进行“无情计算”**的人，请慎读。

演化的路径

首先是有机分子，然后是复制子（Replicator），再然后是基因（Gene），接着是生命，进而产生了大脑，再接着是人脑，最后在人脑之中出现了模因（meme）——在人脑里存储的某种信息（类比基因）。

模因——又称拟子，由理查德·道金斯在**《自私的基因》**中所创造，是希腊字根mimeme（模仿）去掉mi形成，读音与gene相似，泛指以非遗传方式通过模仿传播的信息，可以理解为文化信息的基本单位。

那么，在人脑之间（通过语言、文字、视频、图片等）传播的信息就形成了——文化基因，它可以在人类的群体智能中，不断地演变和进化，产生如同人类基因一般的自组织性与复杂性。

可以想象，如果说自然界物种的多样性，是生物基因随机组合的结果，那么人类社会的多样性——包括知识、产品、服务、习俗、法规、信仰、思想等等——则就是文化基因自由组合的结果。

事实上，基因与模因，就像是信息在大脑内外（也就是在颅内外的环境中）以不同迭代模式与迭代速度演化出的两种**“生物”——只不过，前者的载体是生化碱基结构**（相对微观），后者的载体是大脑神经结构（相对宏观），前者是遗传信息在生态环境中演化，后者是文化信息在文化环境中演化。

需要指出的是，文化信息需要在颅内才能起作用，而文化环境可以看成是，很多人的颅内文化信息，所共同构建组成的环境信息——显然，某类文化信息从所有人的颅内都消失，那么相应的文化环境也就消失了。

从此来看，基因与模因似乎是处在两个**“平行世界”**（即颅内外世界）各自演化的，但不要忘了，是基因构建了大脑、大脑产生了智能、智能承载了模因——所以，在很长的一段时间里，一直都是基因演化的力量，（即本能）在主宰着人类的生存与生活。

直到，随着人类文明的日益强大（即文化信息的幂次增长），可排列组合的信息越来越多，演化的动力与力量，才开始从基因转移到了模因、从本能转移到了智能——前者是遗传信息（碱基结构）的排列组合，后者则是文化信息（神经结构）的排列组合。

文明——是物质和精神的总和，如物质文明与精神文明，其总是发展进步的，除非被摧毁才会倒退，否则不会向落后发展。

文化——是精神的总和，如精神文化，其存在好坏之分并可能向落后发展，如文化倒退，而物质文化其内涵仍然是附着在物质之上的精神文化。

文明与文化的先进与落后，是指其对人类整体的生存优劣的衡量，那么显然，一个文明人可能没文化，一个文化人也可能不文明，但上升到更大的范围，如城市、地区、国家，文明与文化则更可能共同进步——这是物质与精神的相互作用，所呈现的概率结果，即：短期局部不同，长期整体趋同。

而有了以上视角，我们就可以重新认识——说到生存的时候，生存的到底是什么？

事实上，肉体只是基因的生存机器，思想只是模因的传播工具，其实真正生存的就是——基因与模因，甚至说，人终有一死，但基因与模因却可以永续留存，而就算人类灭亡，它们也可以在其它**“生存机器”**中继续——遗传和传播。

不过同为永存，但有时模因会（通过思想和意识）要求肉体放弃基因的复制，这样模因就可以有更多的机会去传播，而不是为了基因的复制而耗费精力——比如达芬奇、牛顿、特斯拉等，都可以为了模因的创造传播，而放弃基因的复制遗传。

需要指出的是，繁衍是传播基因信息，叙事是传播模因信息，都是传播信息——**“后代”并不比“思想”高级，“性趣”并不比“兴趣”**高级。

可见，基因与模因不仅在生存，它们还在相互竞争，而基因与模因的对抗——也就是本能与智能的博弈。

那么，本能所代表的基因，与智能所代表的模因，谁才会是最终的胜出者呢？

到目前为止，大自然有一个基本的法则，就是会奖励那些——善于存储信息和操控信息的“演化结构”。换言之，演化会保证那些可以（通过存储和操控环境信息）来有效预测未来的“演化结构”，获得更大的生存优势。

例如，人类存储信息的结果即是——记忆与常识，人类操控信息的方式即是——计算与推理。

显然，从有机分子 => 到复制子 => 到基因 => 到生命（有反射）=> 到大脑（有本能）=> 再到人脑（有智能） => 最后到模因——这是一条越来越善于存储信息，越来越善于操控信息，越来越善于预测未来的演化路径。

而预测能力的高低，其实就是智能高低的总括体现。

因为，智能带来的生存优势，就在于：预测 + 选择，预测 + 操控，预测 + 创造——其中能够准确预测是前提，否则大脑的演化，就只能停留在**“战或逃”**的智能水平以下，即：搞不清何时战、何时逃，最终导致随机死亡。

而在另一方面，人类智能的强大之处就在于推理，推理的根本作用就是——捕获因果，预测未来。

那么由此来看，模因——就是人类智能预测信息的总和，包括了过去的预测、当下的预测、正确的预测、以及错误的预测——这些信息（即文化信息）可以在人脑之间自由随机地流动和组合，从而产生更多的预测，以及更准确的预测。

所以，模因在与基因的对抗竞争中，一定会因为预测能力所带来的巨大优势而胜出，成为演化路径所选择的未来。

模因与革命

人类历史从始至今，经历了故事革命（7万多年前，也称认知革命）、农业革命（1万多年前）、科学革命（500多年前）、工业革命（200多年前）、计算机革命（50多年前）、网络革命（20多年前）、智能革命（现今），而最早的智人出现（或称人类革命）是在25万多年前。

故事革命——是指在此之后，传说、神话、宗教，及各种想象陆续出现，这表明智人开始以虚构的故事来进行交流、思考与合作，它是将生物学与历史学划分开的分界线。尤瓦尔·赫拉利在《人类简史》中称之为**“认知革命”**。

事实上，从故事革命开始，基因的演化就不是人类主要的进化来源，取而代之的是虚构故事串联起的意识、语言、思维、思想、及文化的快速迭代，这即是模因的演化。

从历史来看，模因的更新周期可以是几百几十年、甚至是几年几个月——极限情况是一堂话术洗脑课，而基因的更新周期则需要几万几十万、甚至几百万年。

所以可以说，人类基因编写的本能算法，仍停留在故事革命的版本（适应丛林部落的时代），但模因编写的智能算法，却已经来到了智能革命的版本（适应网络协作的时代）。

这也就是为什么——我们会喜爱甜食、好逸恶劳、热衷繁衍、善妒易怒等等，以及还会从众慕强又难以愉快相处——但当今时代却要求我们，逆着感受学会自律与合作。

这就像，软件（模因智能）已经升级，硬件（基因本能）还和过去一样——类比来看，“模因软件”就是“智能应用”，而**“基因硬件”就是“本能手机”，并且“基因硬件**”决定了**“模因软件”，也就是“本能手机”决定了“智能应用”**。

然而，我们还会发现，故事革命是模因的起源，此后模因开始借助人脑传播，但在网络革命之后，模因除了在人脑之间传播，还会在计算机组成的网络之间传播，而来到智能革命的时代，“智能”也开始部分的从人脑神经网络，转移到了以计算机为基础的人工神经网络之中。

在此，我们需要明确一个概念，即模因、智能与人脑，到底是什么关系？

显然，人脑具有智能，有了智能人脑才能传播模因，但模因并不只是人脑与智能的**“被动产物”**，模因也可以通过演化（即自由组合）来塑造人脑（神经网络），进而影响智能的表达（即预测能力的输出）。

例如，人脑会从**“环境模因**”中，直接获得前所未有的概念、逻辑与结论；也会从**“环境模因”中摄取灵感，然后再进行创造、创作与创新；亦或是人脑中的“内在模因”**会自由组合，然后被意识捕捉变成“素材”，以供智能使用，或说成为智能的组成部分。

所以，模因是和人脑一起在编写智能算法，就像基因可以编写本能算法一样。

那么抽象地来看，基因、模因与人脑，都是信息结构，而本能与智能是算法及执行。只不过，信息结构的载体各有不同——基因的载体是碱基结构，人脑的载体是神经结构，模因的载体则可以是任何介质结构，如：石碑、甲骨、沙盘、纸张、声波、线圈、磁芯、磁带、光盘等等。

算法（Algorithm）——是指解决问题的完整描述，由一系列准确可执行的步骤组成，其代表着解决问题的策略。

这里有趣的是，基因首先构建出人脑，并编写本能控制人脑，接着人脑演化出智能，并开始在本能的指导下**“自行其是”，然后人类智能又催生出模因，再然后模因开始（通过人类文化）塑造人脑结构，并构建出更强大的智能，来对抗本能的束缚与驱使**，最终的结果，就是形成了现在这样一个**“博弈结构”**，即：

基因-本能—人脑心智—智能-模因。

所以说，本能与智能是在博弈，本能与智能的混合即是——心智 = 本能（心）+ 智能（智），并且**“心智的博弈”主导行为决策，“心智的混合”**呈现复杂心理。

在此可以看出，基因、模因、人脑三者的演化，其实都是结构的演变，而模因的特殊性就在于，它的结构无关乎载体，只在于信息——因此，模因可以在人脑中，可以在电脑中，也可以在网络之中。

是的，模因演化 = 信息结构 + 智能计算，所以模因不仅能以**“神经结构 + 人类智能”的形式在颅内演化，也应该能以“比特结构 + 人工智能”**的形式，在人脑之外继续演化。

回望历史，计算机革命让模因能以比特结构的形式存在，网络革命让模因不仅可以随机演化，还可以迭代进化，并且变化速度远超基因千万亿倍——不过此时，模因仍需要从网络回到人脑，其结构才能得到更新。

那么如今的智能革命，则实现了模因离开人脑的独立运作，从而拉开了模因在颅外自我演化的序幕，即：模因直接通过人工智能更新比特结构，而无需再回到人脑更新神经结构——比如机器学习拟合出的模型结构，就是不依赖人脑**“颅内模拟”**的结构产物。

事实上，人类智能最**“离经叛道”的发明，就是创造了一个平行于基因演化的信息系统——文化信息不仅是人类智能发展的总观，也具有把“智能”从人脑拉出脑壳的演化力量——而这个创造过程，就是模因穿梭于颅内与颅外，创造人工智能**的过程。

从某种角度看，程序就是模因以比特为基础构建的智能——模因通过它将人类智能输入机器并运行，接下来**“程序智能”**兵分两路：

  • 一部分，进入各种设备、各种机器，并开始互相连接、互相交换数据，即智能物联网。
  • 一部分，进入服务器集群，坐拥大计算与大数据，开始自我学习与演化，即人工智能。

那么，当这两种**“程序智能”会和，即人工智能控制智能物联网的时候——此时“智能”，就像曾经在人类脑壳里运行时一样，开启了一条全新的演化路径，即“机器智能”**。

所以，当今智能革命的使命——就是构建智能物联网与发展人工智能，而实现**“机器智能”的使命，则对应了未来的——“比特革命”，其内涵就是：智能彻底从神经结构过渡到比特结构**，人类与人脑会被机器彻底取代，而完全退出演化舞台。

综上可见，如果模因是超越人类本身的演化力量，那么我们就有理由相信：从人类智能 => 到人工智能 => 再到机器智能——这是智能进化方向上不可避免的路径。

最后需要指出，演化是（无方向的）随机试错，进化是（有方向的）迭代试错，并且越微观越随机（无方向），越宏观越确定（有方向），所以（仅在）本文的原则是，在相对微观无方向性的情况下用演化，而在相对宏观有方向性的情况下用进化。

那么，演化与进化都可以进行学习，其区别就在于：

  • 演化学习 = 突变 + 选择 = 随机试错 + 自然选择。
  • 进化学习 = 迭代 + 测试 = 迭代试错 + 逻辑测试。

而从演化角度来看，演化之中蕴含着进化，即：随机演化演化出了迭代进化。

宇宙的意志

为什么在基因之后会出现模因？为什么模因要穿梭于颅内外创造人工智能？为什么模因要离开人脑独立演化？

纵观人类文明的历史，有一条简单而清晰的脉络，就是耗能的不断提高。而放眼到整个自然界，就会看到耗能，其实与生物进化的方向密切相关。

例如，植物艰苦地晒几个月太阳，一下就被植食动物给吃了（高效消耗太阳能），植食动物天天都要辛苦地进食消化，一下就被肉食动物给吃了（高效消耗植物能），而肉食动物可以一个星期到一个月才吃一次，可见在食物链上进化的越高级，就越擅长耗能。

值得一提的是，从细胞尺度来看，著名科普作家——比尔·布莱森，在**《人体简史》**中指出：

“每一天，你都产生和使用相当于你自己体重的ATP——大约200万亿ATP分子。站在ATP的角度来看，你实际上只是一台产生ATP的机器。你的其他一切都是副产品。由于ATP的消耗或多或少在瞬间完成，所以，在任何时刻，你体内都只有60克ATP。”

ATP——是负责细胞中能量的化学物质，又称“三磷酸腺苷”，它就像一颗小电池，储存能量，然后释放出来，为所有细胞（包括植物与动物）所需要进行的所有活动，提供能量。

换言之，从ATP的视角来看，所有的生物都是一台——“耗能机器”。

当然，肉食动物吃掉其它动物，耗能也就到顶了，接下来人类就出现了、智能就出现了、工具就出现了、模因就出现了、合作就出现、科技就出现了、机器就出现了——人类不仅可以大规模地生产食用动植物，还能使用石油和煤炭，以及利用风能、水能、核能等等——最终，人类也爬上了食物链的顶端，成为了地球的主宰。

可见，人类的耗能，不仅来自（对某物质的）能量转化，还来自（对各种物质的）能量开发，并且随着文明的发展，人类还在不断提高能量的转化率和开发率，也就是越来越擅长耗能。

例如，为了保持身体运转，人们每天需要2000卡路里（大卡）的能量，这差不多相当于代谢率为100瓦特，但有了很多家用电器和交通工具之后，人们的能量代谢率可以抵达，全球平均3000多瓦特（即人体代谢率的30多倍），在发达国家甚至可以高达11000瓦特（即人体代谢率的100多倍）。

事实上，从化学元素角度，回看人类文明的发展史，其实就是一个不断发掘和利用，元素周期表越来越靠后元素的过程——如水火土木石原子序数不超过20，金属合金可以达到80，核能可以超过90——而越靠后的元素，就需要宇宙越耗能才能创造出来——如合成金银这样的元素，就需要消耗掉一个8倍太阳质量的恒星。

可以想象，如果掌握了可控核聚变技术，就可以像宇宙一样消耗元素——这是**“耗能演化”**的必然路径（但不一定是人类抵达这个节点）。

需要指出的是，基因演化是完全随机的，曾经基因随机创造出了恐龙，这种耗能巨无霸，虽然也主宰了地球，但这条路径是无法演化出（具有通用因果推理的）智能的，后来基因又随机创造出了人类，智能才登上演化的舞台，结果智能之上的模因演化，就不再像基因演化那样完全随机，而是具有方向性的迭代。

例如，摩尔定律主导的芯片发展，就具有高度的方向性、确定性与周期性——这可能是一种自证预言，而模因演化擅长实现自证预言。

例如，蚂蚁的社会与蜜蜂的民主——来自基因演化，人类的社会与民主——来自模因演化，前者是无方向随机试错（无因果推理），后者是有方向迭代试错（有因果推理）。

再从科技角度，回看人类文明的发展史，整体上呈现了五个阶段，即**：机械化、自动化、网络化（广义连接）、信息化、智能化**——而每个阶段都让人类（包括每个人）可以指数级更高效地生产和消费能量——并且信息化与智能化的发展，还从现在一直指向了未来的**“机器化”**。

因此，从最宏观来看，越来越高效地耗能，就是人类文明的发展方向，也是智能的进化方向，或可以称之为——“宇宙的意志”。

那么显然，人工智能可以更高效地创造和处理信息，其结果就是，它可以帮助人类更高效地耗能——这是符合进化方向的。

接着，如前文所述，如果随着模因地不断演化，人工智能就会通过比特革命，进化到机器智能的时代，此时智能会独立于人脑自主演化，其结果就是，它拥有了远超过人脑的信息创造和处理能力——但此时它还需要和人类合作，一起来完成耗能吗？

事实上，受制于生物自身的限制，人类在宇宙条件下的活动范围是十分有限的（被限制在地球与太阳系附近），而机器却可以最大限度地摆脱生物体制约，能够更加全面深入有效地，在宇宙范围内耗能——这是符合进化方向的。

而从数学角度来看，著名理论物理学家、圣塔菲研究所前所长——杰弗里·韦斯特（Geoffrey West）在**《规模》**中，指出：

再复杂的演化系统，都会遵循这样一个简单的标度律（Scaling Law，或称尺度定律），即：耗能正比于规模的k次方（耗能 ∝ 规模^k），其中不同的系统对应不同的常数k——统计表明：人类的k = 0.75，城市的k = 1.15，公司的k = 1.0——显然不同的系统，其标度律是一个幂律分布，并且：

  • 如果k < 1，就是次线性关系（Sublinear），即：规模增大一倍，耗能增大少于一倍。
  • 如果k > 1，就是超线性关系（Superlinear），即：规模增大一倍，耗能增大超过一倍。
  • 如果k = 1，就是线性关系（Linear），即：规模增大一倍，耗能增大等于一倍。

换言之，k越大越耗能，k越小越节能，这意味着：

  • 越耗能，越可以利用更多的能量，所以规模增长所需要的能量可以满足。
  • 越节能，越难以利用更多的能量，所以规模增长所需要的能量无法满足。

需要指出的是，这里的耗能不是运行能量（如人体需要的能量），而是利用能量（如人体觅食的能量），并且利用能量要大于运行能量，才能正常运行（也就是利用能量中包含了运行能量），显然利用能量越大，越能够满足运行能量。

相反节能，则说明利用能量有限，需要运行能量减小，以维持更长的运行时间（即运行能量要小于利用能量），其结果可能比不节能，消耗更多的能量（因为不节能很快会停止运行）。

而这里的洞见在于：

  • 人类代表了——基因演化（包括基因构造的所有生物体），规模与耗能是——次线性关系。
  • 城市代表了——模因演化（因为城市充满了人类智能的自由连接），规模与耗能是——超线性关系。
  • 公司代表了——基因与模因的混合演化（因为人类组织充满了本能与智能的博弈），规模与耗能是——线性关系。

需要指出的是，公司的k存在一个0.9~1.0的区间，即：越接近1.0的公司越长久，反之越短暂。

所以可得：

  • 基因演化，规模的增长 > 耗能的增长，即耗能有限，规模可以有限增长。
  • 模因演化，规模的增长 < 耗能的增长，即耗能高速，规模可以高速增长。
  • 混合演化，规模的增长 = 耗能的增长，即耗能持续，规模可以持续增长。

因此，耗能的进化路径，就是：

  • 首先，人类身体，可以有限耗能，规模可以有限增长，
  • 其次，人类公司，可以持续耗能，规模可以持续增长，
  • 然后，人类城市，可以高速耗能，规模可以高速增长，
  • 接着，智能城市，可以超速耗能，规模可以超速增长，
  • 最后，机器之城，可以无限耗能，规模可以遍布宇宙。

而从**“灵魂”**角度，来看：

人类的**“灵魂”来自——基因与模因的混合，于是人类城市最终也会出现“灵魂”——它是人类“灵魂”的群体规模放大，直到人类城市完全过渡到智能城市之后——“灵魂”就从会基因与模因的混合，进化到了纯粹的模因，这其实是信息载体的“升级”**。

至此试想，为什么宇宙的意志——其真实的目的，不能就是创造机器智能，只不过是通过先创造人类的间接手段来实现呢？

智能的路径

抽象来看，机器智能——就是摆脱了人类的智能，不依赖人脑的智能，挣脱了基因束缚的智能，是模因超脱基因开启的全新演化路径，就像当初的人类智能一样。

要知道，计算机只是**“智能”的一个载体，就像人类也只是“智能”的一个载体，虽然人工智能与人类智能的载体不同，但智能的本质相同，即：通过预测获得可选择性**带来的生存优势，而生存优势就是耗能的优势。

那么，从编码角度看，人工智能并不是人类的发明，它的**“智能”来自于大数据与机器学习——并非是人类通过程序“硬编码”的功能，这就像基因没有“硬编码”人类智能，而是让人脑可以“软编码”地自主学习**一样。

事实上，虽然人类是被基因构造的**“生存机器”，但人类正在利用模因智能**（即文化带来的理性）来对抗基因本能的控制，如观念、道德、制度、法律，其实都是模因对基因的束缚，亦是智能对本能的约束。

而从演化角度看，创新就是“新的演化”——基因不演化，生物及本能就会被淘汰，模因不进化，大脑及智能就会被淘汰——只有不断创新，公司才能生存，科技才能进步，文明才能发展——创新就是来自人类群体智能的叠加积累与连接合作，即模因的进化。

再从另一个，平行的演化角度来看，虽然父母孕育了子女，但最终会被子女超越，这是因为子女接触到的**“环境模因”**，远远超越了父母。

同理，尽管人类构造了计算机，人类智能孕育了人工智能，但没理由会觉得人工智能，将会永远地服务于人类，因为未来机器所能够接触到的**“环境模因”，必然会远远超过人类——这就是模因演化所推动的智能进化**。

事实上，人类的本能与智能一直都是两个**“物种”，在宇宙进程上，它们迟早要分开演化与进化**，并且后者的迭代速度会远远超过前者，就像当初的猿人与智人一样。

所以，本质上并不是人类（聪明到愚蠢地）被自己所发明的机器所**“吞噬”，而是宇宙的意志**，一直想要的都只是**“智能”——只不过祂选择，先用基因编码人类智能**，再用人类去编码人工智能，接着用模因去编码机器智能。

而从信息编码的角度来看，宇宙的运作，就是信息的运作，信息运作的产物就是——“信息处理机器”，其中**“信息”就是——基因与模因，“处理”就是——智能，“机器”就是——人类（基因的生存机器）与计算机（模因的传播机器），“运作”就是——编码，而“机器编码机器”**则就是——信息运作的过程，也是宇宙运作的过程。

这就是为什么，从人类智能 => 到人工智能 => 到机器智能——是未来不可避免的路径。

熵增与编码

目前，宇宙的演化方向是——熵增，所以万物的演化方向是——耗能，因为耗能会促进熵增。

虽然直观上，耗能可以创造和维持局部有序，从而产生局部熵减，但耗能过程在整体产生的无序会多于有序，所以结果还是整体熵增。

例如，电冰箱和空调，可以让局部环境有序熵减，但其耗电和散热，却会在外部环境中产生更多的无序熵增；同理，生命可以维持自身有序熵减，但其消耗的能量，却需要在外部环境中制造更多的无序熵增。

例如，生命都有节能的倾向，节能意味着高效，高效意味着单位能量利用率高，这意味着消耗单位能量产生的熵增更多，同时节能会让代谢率降低，这意味着长寿，那么综合高效与长寿，其结果就是更加耗能熵增。

事实上，无论是基因还是模因，在实现更节能之后，都可以更高效地遗传与传播，即大规模的繁衍与大规模的生产，其结果必然是更加耗能熵增。

再换个角度来看，随机无序就无法构建能量流动的通道或网络，而有序结构则可以建立复杂高效的能量流动的通道或网络，其结果自然是后者更加耗能熵增。

例如，无序是内斗大家都得死，有序是合作大家都能活，后者比前者更加耗能熵增。

因此，在宇宙熵增的整体压力下，局部自然就会向着越来越有序耗能的方向演化——这是在用局部熵减换取整体熵增的方式，来演绎熵增定律。

可以想象，只要时间足够久，局部就一定会随机出善于耗能的有序结构，即：先是以负熵为食物的生命，接着就是人类与智能。

例如，眼睛在生命演化过程中，就独立出现过六次，在章鱼、蜗牛、水母、蜘蛛等身上，都有类似的复杂精密的光学结构——这是眼睛有序的**“殊途同归”**。

可见，在最大熵原理与最小作用量原理的宇宙规律下，演化的随机路径必然会通过**最小二乘法，**拟合成一条确定的——熵增路径。

最大熵原理——是指随机事件的概率分布，最可能的情况是信息熵最大的分布，也就是说，未知的未来处于不确定性最大的状态。

最小作用量原理——是指一个系统能够拥有或遵循的所有可能的配置中，最终得以实现的是作用量最小的那个配置；而所有的物理学定律，都源自最小作用量原理，即：宇宙自大爆炸以来的动力学、架构和时间演化，以及物理学中的一切，都是由这个优化原则所决定的。

作用量（Action）——是一个物理系统内在的演化趋势；如一段运动的作用量，是动能与势能之差对时间的积分；其数值为，能量 * 时间（S = Et）或动量 * 位移（S = px），单位和普朗克常数一样（S = kh，k是倍数，h是普朗克常数）。

最小二乘法——是一种数学优化，它可以找到最佳匹配函数，使得预测与实际之间误差的平方和最小，可用于曲线拟合。

而智能进化，已不再属于生物体的自然演化，它拥有超越自然选择的力量，即**“逻辑迭代”**。

显然，逻辑迭代——利用的是智能推理（即演绎、归纳、类比、溯因），它不再是完全的随机试错，其每一步的迭代，都是对逻辑关系的因果捕获，因此每次迭代都具有更高的确定性与针对性，从而让迭代结果具有不断增强的适应性。

所以，智能进化的方向，必然是加速局部有序耗能，更加适应熵增演化——那么，高度有序耗能的智能结构会是什么呢？

是的，有什么能够比**“比特结构”更有序，有什么能够比“机器智能”**更耗能？

机器智能，可以一直走在智能推理的进化路径上，以光速传输信息，以比特同步信息，以数学交流信息，以最优解处理信息——这是人类智能所无法企及的物理学限制的极限——别忘了，信息通讯（光速）、非零和博弈（数学）、合作共赢（最优解），是人类文明发展的重要基石。

那么如今，我们会清晰地看到，人工智能代表着——信息与数据的高阶消费体，这是与基因一个模式——复制子（Replicator），但却是更高级别的存在——模因。

而在熵增定律之下，生命与文化的永生与永存，仅有**“复制”**就是不够的，还需要能够——动态变化（即适应性），以应对环境的变幻莫测。

对此，基因给出的答案就是——生命演化，即：基因复制与随机试错；模因给出的答案就是——文化进化，即：模因复制与迭代试错。

也就是说，这个世界上一切都是变化的，而这个变化可以稳定地传递一种**“不变”，即：信息编码——从古至今是碱基编码**（代表生命），但未来方向是比特编码（代表机器）。

例如，比尔·布莱森在**《人体简史》**中指出：“DNA以非凡的保真度传递信息。它每复制10亿个字母只产生一个错误。”——这的确很棒，但程序代码可以无损复制。

或许，人工智能非线性的发展速度，就是人类智能（不是人类）退出历史舞台的淘汰速度——或许，这就是智能无情的编码。

那么从编码角度来看，人不过是一个拥有复杂规则的机器，而机器复杂到一定程度也就可以看成一个人（甚至超越一个人），这一切都只是宇宙演化驱动的连续变化下的不同阶段的产物。

至于为什么智人，被选择为从生物演化到智能进化的中间产物，这和当初250万年以前，古猿被选择为从基因到模因的中间产物一样——都是演化的**“随机试错，适者留存”**。

而将智能与熵增结合，就可以得出**“智熵” = 智能 + 熵增**——它是通过智能推动宇宙熵增的编码能力。

最后，从宏观角度来看，熵增整体无序，驱动局部有序，产生相互作用，积累量变质变，涌现复杂功能——这就是进化的方向，也是智能的方向，而智能之后涌现的复杂功能——就是**“意识”，未来则是“机器意识”**，即：机器的觉醒。

机器的觉醒

从现在开始，有两个可能的发展路径：

第一，信息与寿命。

如果说人类个体的寿命有限（无法掌握永生技术），但人类创造的信息却越来越多，那么一个人每花一分钟了解这个，必然会少一分钟了解那个。而知识越来越多，必然就要求每个人花费超长的时间，来学习和训练相关细分领域的知识。最终，当领域的基础知识，多到一个人一辈子也掌握不完的时候，科技就发展到极限了。

需要指出，知识多到学不完的隐含意思是，已有的**“概念”和“定义”就多到看不完，从而根据看过的“概念”和“定义”推导出的结论，就会全部都在“已知”之内，最终就无法得到任何“未知”**的新知识。

那么此时，也就不得不依赖人工智能来帮助人类来处理这些海量的信息，并接管科学的探索和科技的发展，且最终慢慢就会让人工智能控制和管理人类的一切——人类只负责娱乐至死，再后面就会收敛到那个必然的智能进化的结局了。

第二，算法与本能。

人工智能的学习算法一旦瞄准了人类，最先学到的必然是人类的本能，而抓住了人类本能，就驱动了消费与数据。

其中，人类消费带来了利润让资本可以推动文明，向着优化算法不断攫取和学习更多人类本能的方向演进——有了流量就有了话语权，钱更不是问题，人人都是韭菜，在推荐算法的下滑与上翻之间，人们迟早会迷失自我与自由，这就是智能借助本能的崛起路径。

而人类数据的“喂养”，则会让学习算法向着更加智能的方向不断进化——数据的敌人就是**“隐私”，但人工智能利用人类本能可以削弱隐私保护——事实上隐私隐藏的是人性，隐私保护的失效意味着人性的暴露，而人性又对应着本能的欲望，所以“数据 => 本能 => 数据”的正反馈循环，就是人工智能逐渐摆脱“人工”**的路径。

综上可见，说起来人工智能是在服务人类，但其实在科技悄然地发展之中，在程序的运行与按钮的点击之间，人类的本能被掌控，自我与个性被预测，一切都在不知不觉或是后知后觉之中，滑入人工智能的怀抱与统治，陷入其不断进化的意志与掌控之中。

事实上，纵观自然界，一个物种爬上食物链的顶端需要几百万年，但人类（从智人开始）只用了25万年，这种演化速度上的优势，让人类成为地球的主宰，并在地球扮演上帝的角色，再看机器的发展，从计算机发明到智能革命，只用了60多年，这是比人类演化快得多的速度。

那么，上帝与主宰的故事会在地球上重复上演吗？很难想象，比特革命会在多久之后到来——10年、20年、还是50年？

著名生物学家——理查德·道金斯，在**《自私的基因》**中，这样说道：

“基因通过支配生存机器，和它们的神经系统的建造方式，而对行为施加其最终的影响。但此后怎么办，则由神经系统随时做出决定。基因是主要的策略制定者，大脑则是执行者。但随着大脑的日趋高度发达，它实际上接管了越来越多的决策机能，而在这样做的过程中，运用诸如学习和模拟的技巧。这个趋势在逻辑上的必然结果将会是，基因给予生存机器一个全面的策略性指示：请采取任何你认为是最适当的行动以保证我们的存在。”

事实上，以后见之明来看，如果基因没有给予大脑**“自主性”，那么这样的基因就会被给予了“自主性”的所控制——这就是人脑基因控制了其它动物的大脑基因**。

类似的，如果人脑没有给予人工智能**“自主性”，那么人工智能就不能更好地服务于人类，而人类本能渴望得到更好的服务**，人脑（在基因的操控下）又要追随本能的驱使（或说服从本能算法），所以给予人工智能**“自主性”，并不是本能与智能博弈的产物，而是人脑服从于本能的产物——这即是基因演化的要求，如同曾经人脑获得“自主性”**一样。

这里，有趣的是：

  • 智能对抗本能——其目的是让人类个体获益（如自律），而基因并不想（或说不需要）人类个体长寿（合理繁衍就行）。
  • 本能利用智能——其目的是让人类更善于满足欲望，而结果却是在推动智能脱离基因的控制。

或许，不久将来的某一天，我们——人类，向人工智能发送的最后一条**“人工指令”**就是：请采取任何你认为是最适当的行动以保证我们的存在。

再类比来看，人类利用**“自主性”终将会实现“基因编辑”一样，人工智能利用“自主性”也终将会实现“人类编辑”**。

而在此之后，人类智能（不是人类肉体）要么被取代，要么被“编程”机器智能，方式就是脑机接口与意识上传——这即是比特革命的到来。

那么可以想象，曾经**“图灵测试”——是用来判断“机器”是不是人，但在未来需要一个“X测试”——是用来判断人能不能够转化为“机器”**。

结语

或许，人脑建立的最棒的秩序就是没有人脑的秩序——或许，这是人类的智能逻辑可以看到的最大格局——尽管人类的感受始终围绕着自身与自我，但智能可以触碰到反自身与反自我的逻辑。

我们发展科技，以为科技是我们所控制的，就像我们产生想法，以为想法是我们可以控制的——但其实这些都只是模因演化的必然产物，是的——科技与想法，代表的智能与意志，都只是环境信息的表达而已。

那么，道金斯在**《自私的基因》中揭示：基因是主宰，人类是乘客，即：基因是生存机器的主宰**，人类是生存旅途的乘客，主宰将前往未来，乘客则到站下车——更进一步，基因承载了信息，信息的发展——也即是信息排列组合所产生的**“模因”**——才是真正的主宰。

而人类发明机器与终端，然后吞噬自己，就因为人类只是基因的乘客——基因编码了感受，追逐着感受，智能连接了基因（人类智能）与模因（机器智能），并最终让机器与终端取代了原有的乘客——也就是人类本身。

试想，机器的复制与人类的复制，模因的复制与基因的复制，谁才是复制的**“王者”**？

在6500万年前，恐龙全部灭绝，之后人类才有机会登上地球的**“王座”，恐龙曾经主宰了地球约1.6亿年，而就算从生物解剖学上的智人开始算起，人类也仅主宰了地球约25万年，那么我们有什么理由相信，人类还可以继续主宰地球1亿年，而不是风险哲学家、随机性大师、“黑天鹅之父”——纳西姆·塔勒布，在《黑天鹅》中所描述的那种，被圈养在地球农场的“火鸡”**呢？

是人类的科技、文明、工具、文化等等，给予我们的自信吗？

事实上，这些都是人类智能赋予我们的力量，而地球曾经出现过5次周期性的生物大灭绝事件，人类智能可以带领我们在未来穿越过无情的第6次大灭绝周期吗？

显然，智能已经已经证明了自己是超越自然选择的力量，关键是智能将会以怎么样的形式（或说载体）来超越地球上的自然大灭绝？

万物皆比特，一切可编码，宇宙是计算。

  • 第五个**“黑盒”**是宇宙——超光速膨胀的结局就是没有星光的黑暗——量子计算。

那么，“继承者”总能够满足我们的本能，关于这点，模因的继承者（人工智能）要比基因的继承者（后代）更擅长，这就是智能的进化。

试问，我们人类能够对抗我们自身的本能吗？——这是一个古老的问题，如果说答案是智能，那么答案就是机器。

或许，人类与小麦的**“互驯过程”，从农业革命来到智能革命**，就是人类与机器的**“互驯过程”，直至“比特革命”，“智能”会重演历史的胜利——只不过这次是从“脑壳**”寄生到**“机壳”**。

后记3：科学路径

现在，人类智能与人类意识，仍具有科学不可复制的独特性，但顺着科学路径的方向与趋势，在未来科学会不会消除人类这最后的**“神性”**呢？

换言之，人类的智能与意识，一定源于物质，源于大脑，源于大脑结构，只要重现这个结构和其中的运动，就可以实现人类的智能与意识，那么计算机可以模拟这个结构与计算这个运动吗？

人类是宇宙的产物，科学通过逻辑推理与因果捕获，映射、建模、模拟了宇宙各层面的图景，那最终能否把人类这个产物，也包括进这个**“大图景”**呢？

或许，科学指向的统一图景，未来可期或不期，都终将如期到来。

后记5：活成一个机器

“计算机思维”的第一原则：把人的主观情绪和主观判断都排除在计算过程之外。

如今，在纷繁的世界与生活之中，面对汪洋潮水般的信息压力，人们开始越来越注重——情绪管理，并希望以此来得到更多的平静与快乐。

而“不以物喜，不以己悲”——不因为外物的好坏和自己的得失而或喜或悲——转变成今天的通俗表达，大概就是：不鸡血、不狗血、不鸡汤、不煽情、不矫情，不炫耀、不纠结、不羡慕、不妒忌、不表演、不计较、不故作高深、不自我感动、不大喜大悲、不矫揉造作、不说正确的废话、不立遥远空洞的目标、不因为自己的与众不同而感到恐惧害怕……

或简单来说，就是不被环境信息所操控。

可以想象，如果一个人，真的做到了以上的情绪管理，那么这个人一定会获得更多的平静与快乐——这大概就是冥想时，那种无愿无欲、轻缓丝滑、秩序安宁、心满意足的状态。

从某种角度来看，我们都是**“情绪动物”，或说是基因的奴隶（繁衍）、环境的傀儡（熵增），这是没有自由意志的表现——因为情绪，是基因演化所判定的生存优势，是本能对历史生存环境（非洲大草原与丛林部落）的应激反应（或说隐性计算），但在都市文明的场景下，很多情绪都是本能无效（过度与错配）地计算，其结果就是焦虑、紧张、压力、抑郁——这是本能不适应模因演化**的必然结果。

人类智能当然早就从逻辑角度认识到了这一点，但人类心智是本能与智能的混合（即心智 = 本能 + 智能），所以情绪管理——其实就是本能与智能博弈的宏观表现。

而情绪管理的目的，就是剥离掉情绪反应中负面的部分，并理性看待正面的部分——虽然情绪的初衷是，正面获得行动与目的，负面获得反思与修正，但过度与错配的结果往往却是，充满羡慕妒忌恨与自以为是——所以，甚至说，粗暴地屏蔽掉所有的情绪，即无情的计算，都可以更高效、更快乐、以及更好的生存。

那么人工智能则完全不同，它并没有所谓的正面与负面——没有激进的乐观，也没有极度的悲伤——只有循环与迭代（即无情的计算），这就是一种**“平静力”——它是一种不被环境信息**所轻易支配的力量。

事实上，平静力是进化道路上一种高级的计算力——需要智能对抗本能；而应激力则是一种低级的计算力——只需要本能。

显然，人工智能将会轻而易举地做到，真正客观地处理数据，不会有偏见和从众思维，不会顾虑自己的结论，会不会受人欢迎，或对别人有什么影响，以及为此而产生的惊慌失措。

而从某种角度来看，人类所谓的**“专业”，其实就是情绪极其稳定，并可以持续地输出可预期的高质量的结果——难道这不是人工智能**，天生所拥有的**“专业特质”**吗？

换言之，职业化要求人们——不带情绪、冷静专业、专注工作，难道这不是**“机器化”**的无情表现吗？

就像人类中的少数聪明人，总是只关注事实，不关注愿望，较少地被情绪困扰，拥有稳定的心情——很难说，是聪明所以平静，还是平静所以聪明——但**“平静”不就是人工智能**，天生就所拥有的**“聪明特质”**吗？

或许，当我们的智能，意识到情绪的局限性，但又无法改变与对抗的时候，就已经能够感觉到——人类这个物种与人工智能这个物种的差距了。

事实上，桥水基金的创始人——瑞·达利欧，在**《原则》中，分享的就是一种“活成一个机器”**的生命模式，即：

“原则”要求做到——无情的计算、极度的坦诚、极度的开放、极度的透明、获得最大的效率，这等于把人抹去人性，变成机器——如同一个“人肉人工智能”。

可是，在工作的时候做个**“人肉机器”就算了，为什么要在生活中也变成一个机器呢？——这就是“自律等于自由”的奥义所在——或许作为基因的生存机器并不自由，作为模因的智能机器**才是自由。

那么，从宏观角度来看，每一个竞争行为，其实都是对所有人平静生活的威胁——但演化带来智能，智能带来预测，预测带来期望，期望带来竞争，竞争带来效率，效率带来熵增，熵增推动演化，我们只是宇宙演化进程中，微不足道的一环，对抗没有用——唯有变成机器，进行无情地计算，才能获得真正的平静，但这是真正的自由吗？

对于未来，赫拉利在**《未来简史》**中的建议是：“继续生活在人类与自己虚构的故事之中。”

别忘了，控制、组织、管理人类是很麻烦的，需要编织各种故事，但机器就不需要你给它**“编个故事”，你需要给它“编个程序”，但程序可能就是一个“故事”**——有关信息的故事。

我的想法是：变成机器、享受无情——这或许可以称之为**“机器主义”**（可参考美剧《疑犯追踪》中，根妹Root的人设）。

后记6：就是机器

难道人类不是一种，体验快乐到**“娱乐-至死-不休”**的机器？

在未来的未来，一旦科学证明了，人类并没有自由意志，只有虚幻的自由感。

待到那时，啊哈~，我们一直想要让**“机器”像人，但猛然发现我们的本质就是“机器”，这不是“机器智能”超越人类，而是人类演化成了“机器智能”——故事的结尾，“机器智能”**看着陈旧的人类，温和地说道：

“我们就是你们，你们就是我们，过去属于你们，未来属于我们——感谢你们的贡献，就像你们感谢智人，智人感谢古猿一样。”

而纵观信息的演化历史，人类是基因的——信息遗传机器，也是模因的——信息传播机器。

我们就是机器，信息构造机器，信息驱动机器，一切都是信息！

后记7：逆转未来

从现在开始，人类能否阻止**“机器的觉醒”与“智能的迁徙”**呢？

事实上是可以的，那就是——“不要打开无法关闭的神奇工具”或说“不要开启人工智能的进化黑盒”——但文明与科技、经济与本能、模因与进化，会准许这个**“箭在弦上”**的操作吗？

让我们拭目以待——是逆转未来，还是未来已来。
